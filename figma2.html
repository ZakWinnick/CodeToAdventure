CODE for Adventure

Buying a Rivian? Use a referral code and get rewards!

You'll be directed to Rivian's R1 Shop. Code changes every page refresh.

How does it work?

When you use an owner's referral code during checkout of a qualifying R1 Shop vehicle, then take delivery, both the original owner (referrer) and new owner (referee) get rewards!

What are the rewards?

  • 🎁 750 points that can be redeemed in Gear Shop or R1 Shop (1 point equals 1 dollar in credit)
  • ⚡ 6 months of charging at Rivian Adventure Network sites (up to a lifetime limit of three years)
