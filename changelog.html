Changelog

Version 2024.12.11 - December 11, 2024

- Complete refresh of the design language of the site (Thanks for the inspiration, Mark!)

- Simplified some backend code to make the site run more efficiently

- Cleaned up all unnecessary/errant code on every page

- Removed the field for X Username submission on the Submit Code page.

- Revamped Admin console with new design language, updated tables, and persistent login via cookies.

Version 2024.12.10 - December 10, 2024

- Added banner for admin's referral code.

Version 2024.11.13 - November 13, 2024

- Created a new admin dashboard for easier editing/deleting entries for mistakes and spam.

- Admins will now be notified, via e-mail, when a new code has been submitted.

Version 2024.11.7 - November 7, 2024

- Moved development to GitHub for better version tracking.

- Changed versioning numbers for better release notes.

- Made Twitter/X handle not required for code submission (Thanks to Nathan for the suggestion!).

Version 2.0 - September 29, 2024

- The page title was updated to "Code to Adventure - Random Rivian Referral Codes" for better branding and SEO.

- The main layout now uses flexbox for full-page centering and responsiveness across all devices.

- Horizontal scrolling on mobile was fixed by adding overflow-x: hidden to the body to prevent the page from sliding around.

- Increased spacing and margins to improve the layout, ensuring no elements are cut off, particularly on mobile.

- Button size and padding were adjusted for mobile, making it fit properly within the viewport without being cut off.

- The header image is now randomly selected from a directory of images, enhancing the visual experience on each page load.

- The footer was updated with proper links to the creator's website and referral code, as well as a mailto link for administrative contact.

- Updated meta tags for better search engine optimization and social media previews (Open Graph and Twitter tags).

- An API was added to allow external access to retrieve random referral codes programmatically. API rate limit set to 100 requests per minute for fair usage and load management. API documentation is available on the "API Docs" page, detailing endpoints and usage.

Version 1.3 - September 4, 2024

- Added e-mail link to admin@codetoadventure.com.

- Moved the domain to codetoadventure.com.

- Added a description to the homepage to better explain the site's purpose and functionality.

- Updated page titles to reflect new site name.

Version 1.2 - August 29, 2024

- Implemented a duplicate check to prevent adding the same referral code multiple times.

- Updated the submission page to display an error message if a duplicate code is detected.

- Improved user experience by providing immediate feedback on the submission page for errors.
