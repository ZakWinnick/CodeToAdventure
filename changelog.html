Version 2.0 - September 29, 2024

- The page title was updated to "Code to Adventure - Random Rivian Referral Codes" for better branding and SEO.

- The main layout now uses flexbox for full-page centering and responsiveness across all devices.

- Horizontal scrolling on mobile was fixed by adding overflow-x: hidden to the body to prevent the page from sliding around.

- Increased spacing and margins to improve the layout, ensuring no elements are cut off, particularly on mobile.

- Button size and padding were adjusted for mobile, making it fit properly within the viewport without being cut off.

- The header image is now randomly selected from a directory of images, enhancing the visual experience on each page load.

- The footer was updated with proper links to the creator's website and referral code, as well as a mailto link for administrative contact.

- Updated meta tags for better search engine optimization and social media previews (Open Graph and Twitter tags).

- An API was added to allow external access to retrieve random referral codes programmatically. API rate limit set to 100 requests per minute for fair usage and load management. API documentation is available on the "API Docs" page, detailing endpoints and usage.
