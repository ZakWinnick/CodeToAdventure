API Documentation

Welcome to the Code to Adventure API! Below are the details of the available endpoints, rate limits, and usage instructions.

Endpoints


Get Random Referral Code

URL: GET /api/random

GET https://codetoadventure.com/api.php?path=random
{
    "name": "John Doe",
    "referral_code": "ABC123",
    "link": "https://rivian.com/configurations/list?reprCode=ABC123"
}

Rate Limiting

Limit: 100 requests per minute.

Exceeding the limit will return:

{
    "error": "Rate limit exceeded."
}

Response Codes

The API uses standard HTTP response codes:

  • 200 OK: Request successful.
  • 404 Not Found: Endpoint does not exist.
  • 429 Too Many Requests: Rate limit exceeded.
  • 500 Internal Server Error: Server issue.

Usage Instructions

To use the API, send a GET request to the endpoint. You can test it using browsers, tools like Postman, or integrate into applications.

Respect rate limits for consistent access.
