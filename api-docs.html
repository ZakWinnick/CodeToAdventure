API Documentation

Welcome to the Code to Adventure API! Below are the details of the available endpoints, rate limits, and usage instructions.

Endpoints

1. Get Random Referral Code

URL: GET /api/random

Example Request:

GET https://codetoadventure.com/api.php?path=random

Example Response:

{
    "name": "John Doe",
    "referral_code": "ABC123",
    "link": "https://rivian.com/configurations/list?reprCode=ABC123"
}

2. Get All Referral Codes

URL: GET /api/codes

Example Request:

GET https://codetoadventure.com/api.php?path=codes

Rate Limiting

Limit: 100 requests per minute.

Exceeding: You will receive this response:

{
    "error": "Rate limit exceeded."
}
